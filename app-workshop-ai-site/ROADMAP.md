# 应用工匠 AI 迭代路线

## 当前版本：v0.2 AI 接口版

已完成：

- 三栏后台工具型界面
- 行业模板
- 平台选择
- 功能选择
- AI 优先生成产品方案
- AI 接口不可用时自动使用本地备用生成
- 自动生成数据库和接口建议
- 自动生成开发提示词
- 复制提示词
- 下载 Markdown 方案
- Vercel Serverless Function
- Netlify Function
- 本地开发服务器

当前限制：

- 没有用户登录
- 没有生成历史
- 没有真实代码生成
- 没有项目级长期记忆

## v0.3：项目工作区

目标：

- 用户可以保存多个项目
- 每个项目有需求、方案、页面、接口、提示词记录

建议新增：

- 用户登录
- 项目列表
- 生成历史
- 方案版本对比
- 本地存储或数据库

## v0.4：代码生成能力

目标：

- 从方案进一步生成项目文件结构和代码

建议新增：

- 生成前端页面代码
- 生成后端接口代码
- 生成数据库 SQL
- 生成小程序页面结构
- 下载代码包

## v0.5：商业化版本

目标：

- 做成可收费的 SaaS 或服务工具

建议新增：

- 会员套餐
- 支付
- 模板市场
- 行业案例库
- 客户交付看板
- 管理后台

## v0.6：智能体工具链

目标：

- 让智能体能调用外部工具，完成搜索开源项目、生成代码、创建文件和部署等任务

建议新增：

- 开源项目搜索
- UI 生成
- 代码文件生成
- 部署任务
- 项目状态记忆

## 优先建议

下一步优先做 v0.3：项目记忆和生成历史。这样用户能持续完善同一个项目，而不是每次重新生成。
