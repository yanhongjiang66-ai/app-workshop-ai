# AI 学考教练 v0.3

这是面向多类考试的 AI 学习诊断智能体原型。产品不局限于考公和教资，后续可扩展到大学英语四六级、考研、专升本、自考、法考、会计、医学、职业资格等方向。

v0.3 已完成：

- 泛考试题库入口
- 公职类、教师类、英语类、学历升学类、职业资格类分类
- 四六级示例题
- 自主练习流程
- 选择答案、提交答案、查看解析
- 一键把题目带入 AI 诊断
- AI 做题思维诊断
- 错题本和诊断历史
- 回填旧题继续复盘
- 导出学习记录
- DeepSeek API 接入
- AI 不可用时本地备用生成

## 文件结构

- `index.html`：页面结构
- `styles.css`：页面样式
- `app.js`：前端交互、题库练习、AI 调用、错题本和学习记录
- `api/generate-plan.js`：DeepSeek 生成接口
- `local-server.js`：本地/服务器运行入口
- `.env.example`：环境变量示例
- `agent-prompt.md`：智能体提示词
- `ROADMAP.md`：后续迭代路线

## 环境变量

复制 `.env.example` 为 `.env`，填写：

```text
DEEPSEEK_API_KEY=你的DeepSeek密钥
DEEPSEEK_MODEL=deepseek-chat
PORT=8792
```

## 本地或服务器运行

```bash
npm install
npm run dev
```

默认访问：

```text
http://127.0.0.1:8792
```

## 当前限制

- 内置题库还是示例级，不是海量题库。
- 学习记录保存在用户自己的浏览器里，不同手机/电脑不会同步。
- 暂无账号登录。
- 暂无云端数据库。
- 暂无支付和会员系统。

下一步建议做 v0.4：题库数据结构云端化 + 批量导入题库 + 知识点掌握度。
