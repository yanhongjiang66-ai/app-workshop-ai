# AI 学考教练 v0.5

这是面向多类考试的 AI 学习诊断智能体原型。v0.5 的重点是让题库进入可批量导入、可校验、可导出的阶段，为后续云端题库后台做准备。

## v0.5 已完成

- 独立题库文件：`data/questions.json`
- 题库字段标准化：分类、考试、科目、章节、知识点、题型、难度、题干、选项、答案、解析、标签、来源、版权说明
- 分类筛选：公职类、教师类、英语类、学历升学类、职业资格类
- 细分筛选：考试、题型、难度
- 每题显示来源和版权说明
- JSON/CSV 批量导入题库
- 导入前字段校验
- 来源和版权说明必填
- 下载题库模板
- 导出当前题库
- 清空本机导入题库
- 自主练习、提交答案、查看解析
- 一键把题目带入 AI 诊断
- 一键生成 AI 原创同类训练题
- 错题本和诊断历史
- DeepSeek API 接入

## 题库合规原则

题库扩容建议只使用三类内容：

1. 公开合规题源：确认允许使用、转载或商业化展示的内容。
2. 自建题库：自己编写或购买授权后录入。
3. AI 原创同类训练题：基于知识点和题型生成新题，不照搬题干、选项和解析。

不要直接整套搬运未经授权的历年真题。商业化前应对题库来源、授权范围、展示方式做审核。

## 题库文件格式

题库位于：

```text
data/questions.json
```

每道题建议包含：

```json
{
  "id": "cet4-vocab-1",
  "category": "英语类",
  "examType": "大学英语四级",
  "subject": "英语-词汇语法",
  "chapter": "核心词汇",
  "knowledgePoint": "动词辨析",
  "type": "单选题",
  "difficulty": "基础",
  "stem": "题干",
  "options": ["A选项", "B选项", "C选项", "D选项"],
  "answer": "A",
  "explanation": "解析",
  "tags": ["四级", "词汇"],
  "sourceType": "AI 原创同类训练题",
  "source": "来源说明",
  "licenseNote": "版权或使用说明"
}
```

## 环境变量

复制 `.env.example` 为 `.env`，填写：

```text
DEEPSEEK_API_KEY=你的DeepSeek密钥
DEEPSEEK_MODEL=deepseek-chat
PORT=8792
```

## 运行

```bash
npm install
npm run dev
```

默认访问：

```text
http://127.0.0.1:8792
```

## 当前限制

- 题库仍是示例规模，不是海量题库。
- 导入题库当前保存在浏览器本地，还不是云端数据库。
- 学习记录保存在用户自己的浏览器里，不同设备不能同步。
- 暂无账号登录、云端错题本、支付和会员系统。

下一步建议做 v0.6：云端题库数据库 + 管理员后台。
