# AI 学考教练 v0.6

这是面向多类考试的 AI 学习诊断智能体原型。v0.6 的重点是把浏览器本机导入题库升级为服务器题库：管理员可以把合规题库发布到服务器，所有访问网站的用户都能看到同一套题。

## v0.6 已完成

- 独立题库文件：`data/questions.json`
- 服务器题库文件：`data/server-question-bank.json`
- 题库发布接口：`/api/question-bank`
- 管理员口令发布：`ADMIN_TOKEN`
- JSON/CSV 批量导入题库
- 本机导入题库发布到服务器
- 刷新云端题库
- 来源和版权说明必填
- 自主练习、提交答案、查看解析
- 一键把题目带入 AI 诊断
- 一键生成 AI 原创同类训练题
- 错题本和诊断历史
- DeepSeek API 接入

## 题库合规原则

题库扩容建议只使用三类内容：

1. 公开合规题源：确认允许使用、转载或商业化展示的内容。
2. 自建题库：自己编写或购买授权后录入。
3. AI 原创同类训练题：基于知识点和题型生成新题，不照搬题干、选项和解析。

不要直接整套搬运未经授权的历年真题。商业化前应对题库来源、授权范围、展示方式做审核。

## 环境变量

复制 `.env.example` 为 `.env`，填写：

```text
DEEPSEEK_API_KEY=你的DeepSeek密钥
DEEPSEEK_MODEL=deepseek-chat
PORT=8792
ADMIN_TOKEN=你自己设置的管理员发布口令
```

## 运行

```bash
npm install
npm run dev
```

默认访问：

```text
http://127.0.0.1:8792
```

## 当前限制

- 服务器题库仍是轻量文件存储，还不是正式数据库。
- 题库发布后台还没有完整登录、审核和编辑界面。
- 学习记录仍保存在用户自己的浏览器里，不同设备不能同步。
- 暂无会员、支付和运营后台。

下一步建议做 v0.7：账号登录 + 云端学习档案。
