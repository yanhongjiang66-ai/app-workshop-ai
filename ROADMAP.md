# AI 学考教练迭代路线

## 当前版本：v0.3 泛考试题库版

已完成：

- 泛考试产品定位
- 公职类、教师类、英语类、学历升学类、职业资格类入口
- 大学英语四级、六级示例题
- 考研、专升本/自考、职业资格示例题
- 自主练习
- 提交答案和查看解析
- 一键进入 AI 诊断
- 诊断历史
- 错题本
- 标记掌握
- 导出学习记录
- DeepSeek API 接入

当前限制：

- 题库数量仍少
- 题库没有后台管理
- 没有批量导入 Excel/CSV 题库
- 没有账号登录
- 不同设备之间不能同步学习记录
- 没有知识点掌握度图谱
- 没有会员、支付和运营后台

## v0.4：题库系统

目标：

- 从示例题升级为可持续扩充的题库系统。

建议新增：

- 题库数据文件或数据库
- 考试类型、科目、章节、知识点四级结构
- 批量导入题目
- 题目难度、来源、标签
- 单选、多选、判断、填空、主观题
- 题目搜索和筛选

## v0.5：云端学习档案

目标：

- 用户登录后可以跨设备保存学习记录。

建议新增：

- 手机号/邮箱登录
- 用户表
- 答题记录表
- 错题本表
- 掌握度表
- 最近学习任务

## v0.6：知识点过关

目标：

- 从刷题升级成知识点掌握系统。

建议新增：

- 知识点树
- 章节进度
- 薄弱点排行
- 每日复习计划
- AI 生成相似题
- 自动安排复习间隔

## v0.7：商业化版本

目标：

- 面向真实用户售卖。

建议新增：

- 会员套餐
- 支付
- 用户后台
- 运营后台
- 域名、HTTPS、备案
- 使用数据看板
- 题库版权和内容合规管理

## 内容扩展建议

优先扩展顺序：

1. 四六级：词汇、语法、阅读、翻译、写作模板
2. 考公考编：行测、公基、申论、职测、综应
3. 教师类：教资、教招、教综
4. 学历升学：考研、专升本、自考
5. 职业资格：法考、会计、建造师、医考等
