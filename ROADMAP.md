# AI 上岸教练迭代路线

## 当前版本：v0.2 错题本版

已完成：

- 考公、考编、教资等考试场景
- 题目、答案、正确答案、做题思路输入
- AI 诊断做题思维
- 输出错因分类、正确路径、知识点补强、错题本总结
- 浏览器本地保存诊断历史
- 浏览器本地错题本
- 标记掌握
- 回填历史题目
- 导出学习记录
- DeepSeek API 接入
- 本地备用生成
- 阿里云服务器部署兼容

当前限制：

- 没有账号登录
- 不同设备之间错题本不能同步
- 没有正式题库
- 没有知识点掌握地图
- 没有会员、支付和后台管理

## v0.3：云端学习档案

目标：

- 用户登录后可以跨设备保存错题本和诊断历史。

建议新增：

- 手机号/邮箱登录
- 用户表
- 诊断历史表
- 错题本表
- 知识点标签
- 最近学习记录

## v0.4：知识过关

目标：

- 从单题诊断升级为知识点训练闭环。

建议新增：

- 考试类型知识树
- 每个知识点的过关题
- 错题自动归类到知识点
- 掌握度评分
- 每日复习计划

## v0.5：题库和自动出题

目标：

- 用户既能诊断错题，也能主动刷题训练。

建议新增：

- 题库管理
- 随机组卷
- 根据薄弱点自动出题
- 相似题训练
- 训练报告

## v0.6：商业化版本

目标：

- 面向真实用户售卖。

建议新增：

- 会员套餐
- 支付
- 用户后台
- 运营后台
- 课程/资料入口
- 域名、HTTPS、备案
- 使用数据看板

## 优先建议

下一步优先做 v0.3：账号登录和云端错题本。这样用户换手机、换电脑后仍然能看到自己的学习记录，也更接近可以收费的产品形态。
