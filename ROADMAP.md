# AI 学考教练迭代路线

## 当前版本：v0.6 云端题库发布

已完成：

- 泛考试产品定位
- 独立题库文件 `data/questions.json`
- 题库来源和版权说明字段
- 公开合规题源 / 自建题库 / AI 原创同类训练题 三类来源管理
- 分类筛选、考试筛选、题型筛选、难度筛选
- JSON/CSV 批量导入题库
- 导入字段校验
- 服务器题库文件 `data/server-question-bank.json`
- 题库发布接口 `/api/question-bank`
- 管理员口令 `ADMIN_TOKEN`
- 本机导入题库发布到服务器
- 所有用户共享服务器题库
- 自主练习
- 提交答案和查看解析
- 一键进入 AI 诊断
- 一键生成 AI 原创同类训练题
- 诊断历史
- 错题本
- DeepSeek API 接入

当前限制：

- 服务器题库仍是轻量文件存储，还不是数据库
- 题库发布后台还没有完整的登录、审核和编辑界面
- 没有账号登录
- 不同设备之间不能同步学习记录
- 没有知识点掌握度图谱
- 没有会员、支付和运营后台

## v0.7：云端学习档案

目标：

- 用户登录后可以跨设备保存学习记录。

建议新增：

- 手机号/邮箱登录
- 用户表
- 答题记录表
- 错题本表
- 掌握度表
- 最近学习任务

## v0.8：题库数据库和审核后台

目标：

- 从轻量服务器文件升级为正式题库数据库和管理后台。

建议新增：

- 题库数据库
- 管理员登录
- 题目增删改查后台
- 按考试、科目、章节、知识点管理题目
- 题目搜索
- 题目审核状态
- 题目来源和授权附件记录
- 支持单选、多选、判断、填空、主观题

## v0.9：知识点过关

目标：

- 从刷题升级成知识点掌握系统。

建议新增：

- 知识点树
- 章节进度
- 薄弱点排行
- 每日复习计划
- AI 生成相似题
- 自动安排复习间隔

## v1.0：商业化版本

目标：

- 面向真实用户售卖。

建议新增：

- 会员套餐
- 支付
- 用户后台
- 运营后台
- 域名、HTTPS、备案
- 使用数据看板
- 题库版权和内容合规审核流程
