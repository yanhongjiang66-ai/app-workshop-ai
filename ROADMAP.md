# AI 学考教练迭代路线

## 当前版本：v0.5 批量导入题库

已完成：

- 泛考试产品定位
- 独立题库文件 `data/questions.json`
- 题库来源和版权说明字段
- 公开合规题源 / 自建题库 / AI 原创同类训练题 三类来源管理
- 分类筛选、考试筛选、题型筛选、难度筛选
- JSON/CSV 批量导入题库
- 导入字段校验
- 下载题库模板
- 导出当前题库
- 清空本机导入题库
- 自主练习
- 提交答案和查看解析
- 一键进入 AI 诊断
- 一键生成 AI 原创同类训练题
- 诊断历史
- 错题本
- DeepSeek API 接入

当前限制：

- 题库数量仍少
- 导入题库仍保存在浏览器本地
- 没有真正的云端题库后台
- 没有账号登录
- 不同设备之间不能同步学习记录
- 没有知识点掌握度图谱
- 没有会员、支付和运营后台

## v0.6：云端题库数据库和后台

目标：

- 让题库从浏览器本地导入升级为云端管理。

建议新增：

- 题库数据库
- 管理员登录
- 题目增删改查后台
- 按考试、科目、章节、知识点管理题目
- 题目搜索
- 题目审核状态
- 题目来源和授权附件记录
- 支持单选、多选、判断、填空、主观题

## v0.7：云端学习档案

目标：

- 用户登录后可以跨设备保存学习记录。

建议新增：

- 手机号/邮箱登录
- 用户表
- 答题记录表
- 错题本表
- 掌握度表
- 最近学习任务

## v0.8：知识点过关

目标：

- 从刷题升级成知识点掌握系统。

建议新增：

- 知识点树
- 章节进度
- 薄弱点排行
- 每日复习计划
- AI 生成相似题
- 自动安排复习间隔

## v0.9：商业化版本

目标：

- 面向真实用户售卖。

建议新增：

- 会员套餐
- 支付
- 用户后台
- 运营后台
- 域名、HTTPS、备案
- 使用数据看板
- 题库版权和内容合规审核流程

## 内容扩展建议

优先扩展顺序：

1. 四六级：词汇、语法、阅读、翻译、写作模板
2. 考公考编：行测、公基、申论、职测、综应
3. 教师类：教资、教招、教综
4. 学历升学：考研、专升本、自考
5. 职业资格：法考、会计、建造师、医考等
